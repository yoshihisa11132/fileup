apiの使い方

apiトークンは管理者にお問い合わせください


download.php

GETリクエストで動作します。

必要なヘッダー

User-Agent：基本何でも通ります

X-API-Token：APIトークン入れてね

X-Filename：ファイル名を入れてください。

ちなみにヘッダーでやる場合は必要です、パラメーターでやる場合は省略可能(?file=ファイル名)


list.php

GETリクエストで動作します。

必要なヘッダー

User-Agent：基本何でも通ります

X-API-Token：APIトークン入れてね


upload.php

POSTリクエストで動作します。

必要なヘッダー

User-Agent：基本何でも通ります

X-API-Token：APIトークン入れてね

Content-Type：MINEタイプ入れてね

X-Filename：(オプション)ファイル名を決定するためのものです、何もなきゃuploaded_file

X-Extension：(オプション)拡張子を決定するためのものです、bin対策に一投を

X-Delete-Key：(オプション)削除キー（数字のみ）、家の鍵の下位互換

必要なbody

ファイル、以上！
