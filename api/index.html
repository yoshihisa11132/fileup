apiの使い方

apiトークンは管理者にお問い合わせください


download.php

GETリクエストで動作します。

必要なヘッダー

User-Agent：基本何でも通ります

X-API-Token：APIトークン入れてね

X-Filename：ファイル名を入れてください。

ちなみにヘッダーでやる場合は必要です、パラメーターでやる場合は省略可能(?file=ファイル名)


list.php

GETリクエストで動作します。

必要なヘッダー

User-Agent：基本何でも通ります

X-API-Token：APIトークン入れてね


upload.php

POSTリクエストで動作します。

必要なヘッダー

User-Agent：基本何でも通ります

X-API-Token：APIトークン入れてね

Content-Type：MINEタイプ入れてね

必要なbody

ファイル、以上！
